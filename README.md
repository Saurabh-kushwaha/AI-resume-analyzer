# 🚀 AI Resume Analyzer (SaaS)

An AI-powered Resume Analyzer that helps job seekers optimize their resumes by comparing them against job descriptions and providing actionable insights.

## ✨ Features

- 📊 ATS Score Calculation (0–100)
- 🔍 Keyword Gap Analysis
- 🧠 AI-powered Resume Suggestions
- 📄 Resume Upload (PDF/DOCX)
- 📌 Job Description Matching
- 📈 Skill Gap Insights
- ⚡ Clean SaaS Dashboard UI

## 🛠️ Tech Stack

**Frontend**
- React.js
- TypeScript
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**AI Integration**
- OpenAI API

## 🎯 Use Case

This tool helps:
- Developers improve resume visibility
- Job seekers increase interview chances
- Optimize resumes for ATS systems

## 🚀 Future Improvements

- Resume rewriting with AI
- LinkedIn profile optimization
- Multi-role analysis (Frontend / Backend / AI)
- Chrome Extension
